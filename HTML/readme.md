##### 2.HTML编写工具

这又是很多入门童鞋的大问题，知道编程需要学编程语言，那么怎么编程？用什么编程？编程就是写代码，写代码的工具就是电脑，除此之外，啥也不知道。

写代码的硬件工具确实是电脑，但是软件工具就是编译器。编译器会将你写的代码编译成电脑自己能"看懂"的二进制数字，然后电脑用这串数字进行操作达到你想要的效果，呈现在返回的页面或者命令行上。

而且不同的编译器编译的语言不同，编译的文件类型也不同，如HTML文件，一般以'.html'结尾，c++文件一般以'.cpp'结尾，java文件一般以'.java'结尾，python文件一般以'.py'结尾。

前端入门的话，最简单、上手最快的就是HTML编程语言，那么编写HTML的工具是什么呢？HTML语言作为网页基本语言，通用性很高，用的low一点，就是txt编辑器都能编写，用的规范一点，就要用专门的代码编辑器了。

代码编辑器也有很多种，写html的话，推荐使用[vscode(点击此链接下载安装)](https://code.visualstudio.com/Download)，**（这里附上VSCode下载安装教程）**因为它的插件生态十分成熟，而且除了可以编译html之外，其他类型文件也可以编译，作为轻量级免费开发软件，vscode的使用相当广泛，是大多数程序员的心头好。除了vscode，也有其他的编译软件，如sublime、hbuilder、visual studio，还有主要做Java的eclipese和intlli IDEA以及主要做python的pycharm等等。

**(这里附上几种编译器的图片)**

我们文章中指出的HTML均指的是HTML5，跟原先的HTML相比，声明更简便，更**语义化**（就是通过名字就能看出他的功能），功能和开发更加完整方便。

首先，我们需要看懂上一篇文章中最后写出的HTML小实例框架的代表的意义，才能仿照这个做出其他更好的HTML网页，先附上实例的源码：
```html
<!DOCTYPE html>
<html lang="en">

<head>
  <meta charset="UTF-8">
  <meta http-equiv="X-UA-Compatible" content="IE=edge">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>first html page</title>
</head>

<body>
  hello world!
</body>

</html>
```
看代码之前，需要了解几个定义：HTML元素语法，HTML元素，HTML标签，HTML元素属性，

大概构成是这样的：

![2-1](img\2-1.png)

通过规范的HTML元素语法，写出HTML元素，HTML元素由标签和属性构成，其中标签相当于房子的框架，属性相当于房子框架的特点。更具体点就是你的房子有多少根房梁和多少面墙等等由标签决定，房梁承重多少，墙宽高和位置多少等等由属性决定。

现在我们再来细说这段代码的意义：

`<!DOCTYPE html>`：对文件格式的声明，告诉编译器，这个是一个HTML5的文档

`<html>`：HTML页面的根元素，所有的HTML元素都要写在这个元素里面

`<head>`：HTML头部元素，包含文档的元数据，就是对整个HTML文件的定义数据都在这里面

`<meta>`：在<head>元素里面，定义整个文档性质的元素，如编码格式为utf-8(<meta charset="UTF-8">)

`<body>`：HTML页面元素，也叫体元素，所有页面可视化内容都在这个元素里面



值得一提的是，所有的HTML元素都是用标签写的，标签元素分为双标签（<div></div>）和单标签（<hr/>），特别注意单标签是自闭合标签，就是必须加一个 '/' ：<xxx/>这样的写法。早期的单标签没有自闭合，在开发过程中就发现HTML规范不够严谨，现在的HTML5都是用自闭合标签表示单标签，它的含义是<xxx/> === <xxx></xxx>，用来简化双标签的定义，注意没有</xxx>这种写法，'/'要写在标签名的后面。

#### 编程入门学习网站推荐

**写在前面的话**

很多初学者在学习编程的时候总是摸不着头脑，不知道自学入门该参考哪些意见，今天作者就根据自己的亲身体验给大家推荐一些从初级到进阶可以参考学习的网站。

**01** **CSDN**

![3-1](img/3-1.jpg)

作为刚刚入门的初学者来说，国内对于基础知识和进阶引导做的较好的作者个人认为非CSDN莫属了，这个论坛拥有大量的编程技术、生活、行业发展等等程序员和产品相关的知识，并且日活跃中国程序员相当多了。除了在这个圈子学习之外，你还能自己写一些技术博客，打卡每天的学习进度，可以说非常的好用了

**02** **博客园**

![3-2](img/3-2.jpg)

如果说，在国内能有一个对于初学者非常友好能与CSDN并驾齐驱的学习网站，那么作者认为非博客园莫属了。虽然博客园的界面看起来跟七老八十的老大爷一样，但是用久了你就会发现真香哈哈（偷偷说一句，作者大一的程序题不会的都是用CSDN和博客园查的，真的超好用）。

**03** **牛客网**

![3-3](img/3-3.jpg)

前面说了两个知识学习型的网站，这个牛客网就属于刷题网站了。这个网站，除了很多基础知识的题型，还有很多程序员招聘的信息，还有各大公司每次春招秋招的真题和模拟题等等，不管是选择题还是填空题或者是代码题，这个网站可以说是相当全面的了。

**04** **Bilibili**

![3-4](img/3-4.jpg)

很多朋友知道B站是各种二次元番剧和阿婆主的天下，但是其实作者个人认为它也是中国程序员免费学习的性价比最高的地方。很多编程视频和教程都可以在上面找到免费的资源，甚至项目源码等等都可以通过相互交流免费获得，也是相当好用了。

**05** **思否—segmentfault**

![3-5](img/3-5.jpg)

这个网站是作者最近几个月学习前端的时候发现的宝藏网站，针对一些有一定基础的朋友深入学习某框架和某技术的时候所需要的支持。思否的用户量相对CSDN来说要少很多，相比CSDN和博客网，个人认为技术性更强，含金量高。

**06** **掘金—juejin**

![3-6](img/3-6.jpg)

跟CSDN和思否类似，都是论坛型网站，非要说点特色的话，跟知乎比较像，但是要说含金量，只能说中国大佬很多都在掘金，虽然CSDN也有，但是知道掘金的一定是对技术比较有追求的程序员了，相对于CSDN的普遍性，掘金的特点跟对于技术的追求和思否有的一拼。

**07** **GitHub**

![3-7](img/3-7.jpg)

学习完了技术，看了国内的论坛，真要说开源项目哪家强，GitHub非它莫属，虽然国内很多公司用的Gitee码云，因为是国内的服务器，网站加载会快很多，但是，更加多元化的开源项目，在GitHub上面真的是百花齐放，对于开发语言和文化，GitHub真的是包容性极强，很多跨国大佬，甚至说是大型互联网公司的领头人都在GitHub上面拥有一席之地，能在GitHub上拥有社区影响力真的是不管面试什么工作，都相当加分的一件事。



好了，今天的介绍就到这里吧，其实还有很多有趣的编程学习网站，比如stackoverflow等等，但是在这里就介绍这7中实用性相对更强的网站吧，有其他问题和想法的都可以通过公众号留言哦！
